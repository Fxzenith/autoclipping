## 📊 How It Works

```
YouTube Video
    ↓
transcript.py → data/transcript.json
    ↓
[YOU: Run Claude on transcript.json]
    ↓
Claude → data/clips.json + data/review.md
    ↓
[YOU: Review & approve timestamps]
    ↓
extract.js → assets/clip_01.mp4, etc.
    ↓
render.js → Outputs/video.mp4
    ↓
Upload to YouTube Shorts / TikTok / Reels
```

## 🤖 Claude Step (Manual)

When the pipeline asks you to run Claude:

1. Open Claude or use Claude API
2. Provide **data/transcript.json** (read the content)
3. Ask Claude to identify 5-10 high-retention clips

**Example Prompt:**
```
Analyze this YouTube transcript and select 5-10 clips perfect for YouTube Shorts.

Requirements:
- Each clip: 10-25 seconds (ideal ~15s)
- Strong hook in first 1-3 seconds
- Self-contained, minimal prior context needed
- High engagement potential

For each clip provide:
- start (seconds)
- end (seconds)  
- title (one-liner)
- hook (compelling quote from first 3 seconds)
- reason (why it will perform well)

Output as valid JSON only, no markdown, in this format:
{
  "clips": [
    {
      "start": 120,
      "end": 135,
      "title": "Your Title",
      "hook": "Your hook quote",
      "reason": "Why it's engaging"
    }
  ]
}
```

4. Save output to **data/clips.json**
5. Create **data/review.md** with human-readable timestamps and excerpts
6. Return to the terminal and approve

## 🛠️ Key Commands

```bash
# Generate transcript
python python/transcript.py --url "https://www.youtube.com/watch?v=VIDEO_ID"

# Extract clips (requires clips.json)
node scripts/extract.js "https://www.youtube.com/watch?v=VIDEO_ID"

# Render videos (requires assets/)
node scripts/render.js

# Full pipeline (interactive)
node main.js "https://www.youtube.com/watch?v=VIDEO_ID"

# With auto-approval (skip manual review)
node main.js "https://www.youtube.com/watch?v=VIDEO_ID" --approve
```

## 🚨 Common Issues

**Q: "Module not found: youtube-transcript-api"**
```bash
pip install youtube-transcript-api
```

**Q: "yt-dlp not found"**
```bash
pip install yt-dlp
# Or ensure it's in PATH
```

**Q: "ffmpeg not found"**
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt-get install ffmpeg

# Windows
choco install ffmpeg
```

**Q: "Missing data/clips.json"**
→ Run Claude step first, create clips.json and review.md

**Q: "Remotion rendering fails"**
→ First initialize Remotion: `npx create-remotion-app projects/yt-shorts`
